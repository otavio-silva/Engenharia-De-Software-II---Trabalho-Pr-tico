Cat Generator

Random cat generation for free!

Como funciona?

Cat Generator é uma IA geradora de imagens aleatórias treinadas com diversas fotos de gatos. Como estas imagens são aleatórias, a imagem gerada é a de um gato que não existe na vida real.
Este projeto é similar a forma como um cérebro funciona quando ele tenta lembrar a imagem de um gato, ele não será exatamente igual ao gato, mas terá uma semelhança da original.

Este projeto foi criado para a disciplina de Engenharia de Software 2 e tem como o objetivo o aprendizado e entretenimento.
